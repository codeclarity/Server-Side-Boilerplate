server-side-boilerplate by codeclarity

HTML Header Skeleton which includes Meta-data, DublinCore and Geolocation

The Serverside Boilerplate was developed to be used with any standard project that is hosted with Apache. It's original usage was intended for use with the HTML5 Boilerplate but has other purposes as-well. For now, it contains a standard .htaccess file, robots.txt and for HTML file.
